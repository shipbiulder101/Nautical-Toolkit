Welcome to Nautical Tools an application that contains useful tools for anyone in the maritime comunity.
The app has been designed to used by every member of the maritime communtiy from shore staff to yachtmaster to enthusiasts.
The app has been specifically designed to be able to be run offline without an internet connection although certatin features of the app will not work.
App updates will not occur until an internet connection has been established then the app will automatically update.

What each page does:
  • Nautical Converter: a series of useful unit conversions.
  • Tide Calculator: a tool to calculate the height of tide given tide tables.
  • AIS: live ship position data.
  • OpenSeaMap: nautical charts with a variety of base map options as well as day/night line.
  • Weather: weather data from Windy.
Note: sometimes pages may be hidden, to show more simply drag the buttons to move them across.

OpenSeaMap is for reference only and should NOT BE USED FOR NAVIGATIONAL PURPOSES as it is not from a national hydrographic office. It also only covers certain areas.
